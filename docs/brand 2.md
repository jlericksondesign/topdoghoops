# Top Dog Hoops Brand

## Working Name

Top Dog Hoops

## Personality

- Competitive
- Encouraging
- Youth-friendly
- Team-first
- Clear and easy for families to use

## Visual Direction

- Mobile-first interface with strong contrast and simple navigation.
- Primary color direction: bulldog red with neutral backgrounds.
- Accent color direction: gold for achievement moments and challenge energy.
- Rounded but sturdy UI details that feel sporty without becoming busy.

## Voice

Short, direct, and motivating. The app should feel like a coach giving clear next steps, not a complicated stats tool.

## Early UI Notes

- Prioritize quick entry on phones.
- Keep parent approval screens plain and trustworthy.
- Make leaderboards easy to scan.
- Treat badges as moments of celebration, but keep the core app focused on practice.
