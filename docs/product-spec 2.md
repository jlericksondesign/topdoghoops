# Top Dog Hoops Product Spec

## Purpose

Top Dog Hoops is a mobile-first PWA for gamifying basketball shooting practice. Players log shot work, parents approve entries, and coaches/admins can track participation and progress.

## Initial App Sections

- Home: entry point and navigation to primary app areas.
- Player Entry: placeholder for daily shot logging and player progress.
- Parent Approval: placeholder for parent review and approval flow.
- Leaderboards: placeholder for team and challenge rankings.
- Badges: placeholder for player achievements and milestones.
- Admin: placeholder for coach/admin review and configuration tools.

## Target Users

- Players participating in the shooting challenge.
- Parents or guardians approving submitted practice sessions.
- Coaches or administrators managing the challenge.

## Technical Direction

- Next.js App Router with TypeScript.
- Tailwind CSS for styling.
- shadcn/ui for shared interface components.
- Supabase for auth, data storage, and future backend workflows.
- Vercel-ready deployment structure.

## Not In This Phase

- Full data model.
- Authentication.
- Shot submission workflow.
- Parent approval implementation.
- Admin tools.
- Production badge or leaderboard logic.
